## Project Description
The career center plays an important role at Sacramento State University. This important tool helps students prepare and succeed in their major by bringing them together and putting together events and opportunities that are very educational for the students. However, the current career center lacks the features, organization, and management. This makes it very difficult for the students to make the first step of giving the career center a chance. The career center website we are proposing provides solutions to these problems that are keeping the career center away from being great. The goal of our career center website is to provide easy access to events to the user and make it easier for the users to sign up and submit their applications for the desired positions. Moreover, features like account management, efficient search bar, account management, calendar, and notifications will improve the organization and effectiveness of the career center for the user and the management team. All in all, the career center website will provide a user-friendly environment in which it will make career opportunities easily accessible with an engaging user interface and overall support for students in their careers.

## Features
- User account management (create, edit, delete)
- Event search and filtering
- Event sign-up and cancellation
- Notification system
- Calendar and event tracking
- Help page
- About us page
